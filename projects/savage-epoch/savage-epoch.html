Savage Epoch

Version: 0.1.0-alpha

Savage Epoch is a worldbuilding project for my home D&D games. It was made using the Gygax 75 challenge.

The Setting Concept

The world is set from the ashes of the old world. It is inspired by the mythological apocalypse "Ragnarok" from Norse Mythology. There are many inspirations to Norse Mythology, including Mythologically and magically.

The world also contains ancient technology from the old world. This technology is so weird that it borders on magical (and in some cases it is).

Sources of Inspiration

  • Horizon Zero Dawn
  • Dr. Stone
  • Thor Ragnarok (asthetically)
